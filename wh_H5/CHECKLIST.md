## 📞 常用命令

```bash
# 查看后端状态
pm2 status

# 查看后端日志
pm2 logs wh-h5-backend

# 重启后端
pm2 restart wh-h5-backend

# 停止后端
pm2 stop wh-h5-backend

# 重载 Nginx
nginx -s reload

# 建立 SSH 隧道
ssh -L 3306:远程数据库 IP:3306 用户@远程 IP -N -f
```

---

## ⚠️ 注意事项

1. **数据库连接**：确保 SSH 隧道稳定，或使用本地数据库
2. **端口开放**：确保防火墙开放 80、443、8088 端口
3. **文件权限**：确保 Nginx 有权限访问 dist 目录
4. **环境变量**：生产环境使用 `.env.production`
5. **定期备份**：备份代码、数据库和配置文件

## 🎯 快速故障排查

| 问题 | 检查项 | 解决方案 |
|------|--------|----------|
| 页面空白 | Nginx 配置、文件路径 | 检查 root 路径是否正确 |
| API 404 | 后端服务、Nginx 代理 | 检查 pm2 status 和 proxy_pass |
| 数据库错误 | SSH 隧道、.env 配置 | 重新建立隧道，检查配置 |
| 跨域错误 | Nginx 代理、CORS | 确认配置了反向代理 |
| 图片不显示 | 文件路径、权限 | 检查 H5_icon 目录权限 |
